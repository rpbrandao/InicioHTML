O que é HTML?

Voltar

A Hypertext Markup Language (Linguagem de Marcação de Hipertexto) não é uma linguagem de programação e sim, como o nome diz, uma linguagem de marcação ou markup. Ela funciona através de “marcações” chamadas tags ou etiquetas, com o propósito de estruturar e definir a natureza do conteúdo de uma página.

TAGS

Voltar

As tags, ou etiquetas, são as entidades básicas de marcação da HTML. Elas são delimitadas pelos parênteses angulares, os sinais de “menor que” e “maior que” “<” e “>”. As tags servem para estruturar, hierarquizar e demarcar o conteúdo de uma página. A maioria usa uma tag de abertura e outra de fechamento. Algumas tags não possuem conteúdo são consideradas tags vazias.
